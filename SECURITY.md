## Reporting a Vulnerability
If you find a security breach or bug, please send it by email to enesalperenayse31@gmail.com
Reporting a Vulnerability
Omni-X prioritizes security because it operates with high system privileges (Full Access). If you discover a vulnerability or bug, please follow the protocol below:

Direct Contact: Please send your findings directly to enesalperenayse31@gmail.com via email before opening a public "Issue".

Report Details: In your email, specify how the bug was triggered (PoC), on which operating system version (macOS) it occurred, and the risk level.

Response Time: Initial responses will be provided within 24 hours for submitted reports, and a status assessment will be shared.

Responsible Disclosure: We request that technical details remain confidential until the vulnerability is fixed and a new v6.x version is released.

⚠️ Important Safety Notice
Full Access: The /delete command provides full access to system files; always check the target path when using this command.

Kill-Switch: The /panic mode disables the Wi-Fi card at the hardware level; to restore internet connectivity
